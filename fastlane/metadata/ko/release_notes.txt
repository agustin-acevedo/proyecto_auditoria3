새로 업로드한 이미지에서 기본적으로 최적 크기와 품질이 유지되도록 이미지 최적화 설정을 추가했습니다. 내 사이트의 미디어 화면에서 이미지, 비디오 및 기타 미디어를 삭제할 수도 있습니다.

그래도 미디어 지원이 더 필요하신가요? 아직 행운이 남아 있습니다. 이제 사이트 미디어 화면에서 썸네일과 원래 이미지가 더 빠르게 로드되고 GIF 썸네일 애니메이션이 제대로 작동합니다. 또한 미리보기 모드에서 미디어 파일을 선택하고 선택 해제할 수 있도록 버튼을 추가했습니다.

중첩된 블록에서 실제로 "그룹 해제" 옵션을 지원하는 경우 해당 옵션만 블록 편집기에 표시됩니다.  그리고 스토리 편집기에 기본 사진 선택기를 추가했습니다.

iPad 사용자의 미디어 항목 상세 정보 화면을 수정했습니다. 통계 화면의 버그도 해결하여 이제 해당 URL을 클릭하면 사이트가 열립니다.

마지막으로, 이제는 사이트 페이지 목록이 더 빠르게 로드됩니다. 사이트에 수백 개의 페이지가 있어도 상관없습니다. 당연히 원활하게 작동합니다.
